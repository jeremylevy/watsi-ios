<h4 align="center">An iOS app for Watsi</h4>

<br>

**Watsi-ios** is an unofficial iOS app for the nonprofit healthcare crowdsourcing platform Watsi. It was built during my studies.

### Install

To install, run:

```bash
pod update
```

then, build and run with Xcode.
